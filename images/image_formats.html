Image Formats



PNG

A .PNG, or a Portable Network Graphics file, is a image file that supports alpha space allowing for invisible or non-white backgrounds. Supports RGBA but not CMYK. Was made to improve upon the GIF file.



JPG

a JPG file is an image stored with a JPEG compression. Best used for pictures. Has lossy compression.



GIF

a GIF is a bitmap image format. Supports 256 colors per frame. Can be animated or not.
